# Overview

A todo list app where people can add tasks, mark them done, and remove them. Built with React, Go, and PostgreSQL.

# Personas

- User — manages their personal todo list.

# Features

- Add a new todo by typing it in and submitting.
- See all todos in a list.
- Mark a todo as complete or incomplete.
- Delete a todo.
- Completed todos are visually distinct from incomplete ones.
